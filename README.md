## 🛠️ Project Goals & Features
* **Audio Enhancements:** Remastered instrumentals, tuned vocals, and polished soundscapes.
* **Custom Charts:** Fresh charts designed for higher intensity and fresh gameplay mechanics. 
* **Lore Accurate:** Captures the true corrupted/pibby aesthetic while trying out fresh musical directions.

## 📁 Installation Guide (FNF Modding)
If you are integrating these remixes into a custom FNF Engine (e.g., Psych Engine, Codename Engine):
1. **Download** the desired audio files via the links provided above.
2. **Convert** the tracks to `.ogg` format.
3. Place the files into your mod's `mods/songs/airborne/` directory.
4. Replace or rename the instrumental and vocals files to match exactly what your `.json` chart expects.
